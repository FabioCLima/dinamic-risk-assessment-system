# Project Overview

Congratulations! In this project, you implemented a dynamic risk assessment model. Your model is able to predict which of your company's clients are at risk of terminating their contracts. You trained and deployed an ML model that's able to make these predictions. More importantly, you set up the operational aspect of the model, so that data will be automatically ingested, and the model will be automatically re-trained and re-deployed when necessary. You also set up continuous, automatic reporting. By setting up these operational components, you ensured that your model will be able to run successfully in production for a long time to come! You used Python for this project, including a variety of modules for ML and reporting tasks.
